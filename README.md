# Point Doctor
doctor clinic web application.

## Requirements
- install docker version 4.25.2
- install wsl ubuntu
- install node version 14.17.6
- install php version 7.4.24

## Installation



```bash
npm install
npm run dev

```

## Usage

```php
docker-compose up -d

```

## Database Seed

```php
php artisan migrate
php artisan db:seed
```

## Access project

```php
localhost:8000
```

## Access Phpmyadmin

```php
localhost:8001
```
